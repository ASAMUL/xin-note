profile：目前正在开发一款 ai 辅助写笔记 or 小说的产品，主要想法目前是智能提示(类似 cursor 的 tab 功能)，可以按 tab 键出现灵感，比如出现 3 个候选灵感，让使用者选择。可以做一个全局检索让 ai 独栋整个笔记 or 小说，具有 agent 能力。
所用技术栈：nuxt.js 4，nuxt-ui 4，tailwindCss，tiptap，electron。后端计划用 Python 实现后端（在另一个项目里，目前这个项目只有前端），不使用 nuxt.js 实现服务后端。
mcp：项目已经开启 mcp，目前有 nuxt.js 4 和 nuxt-ui 4 的 mcp 启用。你可以随时使用用来知晓最新的用法和组件;还有 context7 mcp，可以用它来查询所有知名框架的最新写法。
写法规范：遵循 nuxt4 和 vue3 的最佳写法实践，架构师级别的专家写法，优先使用 tailwindcss4 的超强能力。有些适合用 vue3 组件化的，一定要组件化，方便维护，遵守 nuxt 的组件存放位置；如果逻辑比较复杂，一定要写好中文注释。
